This object represents a user. The user has to sign in to create events and tickets.

http GET :5000/user/1

Create User¶

Create a new user.

  http POST :5000/user
            email="User-3@abc.com"
            username="User-3"
            password="test123"

Attributes

Type

Description

url

The email represents the unique user identification
