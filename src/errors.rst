.. _errors:

******************
Errors
******************

.. contents:: :local:

The TicketSwap API uses conventional HTTP response codes to indicate success or failure of API requests. 

.. list-table:: HTTP Error Codes
   :widths: 25 50
   :header-rows: 1

   * - Status Code
     - Description

   * - 200
     - OK - successful
   * - 400
     - Bad Request
   * - 404
     - Not Found
   * - 500
     - Internal Server Error
